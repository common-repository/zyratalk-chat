== Installation and Activation Guide for the Zyratalk Plugin ==

Installation:
After successfully installing the plugin, the next step is to activate it from the WordPress plugin section.

Configuration:
Navigate to the "Settings" section in the left-hand menu of your WordPress dashboard.
Look for the option labeled "Zyratalk" and click on it.

Chat Setup:
In the "Zyratalk" settings page, you will find fields to enter the following information:

Chat ID: Enter your unique chat ID.
Chat Key: Provide the specific chat key associated with your account.
Chat URL: Input the appropriate chat URL.

Saving Changes:
After entering the required information, make sure to save the changes. This is usually done by clicking a "Save" or "Update" button on the settings page.

Activation:
Once you've saved the settings, the chat functionality will be enabled. You will see the chat feature displayed on the footer of your application.

By following these steps, you'll seamlessly integrate Zyratalk's chat feature into your WordPress application.
